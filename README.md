# Heart Health Chatbot
An AI-powered Heart Health Chatbot built with FastAPI and Google Gemini AI.
The chatbot provides users with general heart health information,answers health-related questions.
It promotes awareness about cardiovascular wellness through an interactive conversional interface.
## Features
- AI-powered health conversations
- Heart health education and awareness
- Usage of NLP and Secure API Integrations
# Technologies Used
#### Backened Framework - FastAPI
#### Chatbot Intelligence - Google Gemini AI
#### Frontend - HTML,CSS,JavaScript
#### Template Engine - Jinja2
#### Programming Language - Python
